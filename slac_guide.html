SLAC Email Setup Guide

Prerequisites

You must receive a secure email from SLAC IT Account Services that looks like this:

[EXT] TAKE ACTION ASAP - Secure: SLAC Active Directory (Windows) Account Created
From: SLAC IT - Account Services <account-services@slac.stanford.edu>
Subject: Stanford secure message notification

Step 1: Open the Secure Message

Click the secure message link in the email. The content should contain:

  • Your temporary username
  • Your temporary password
  • Account creation details and instructions

Step 2: Set Your Permanent Password

  1. Navigate to:
    https://adfs.slac.stanford.edu/adfs/portal/updatepassword
  2. Enter your username and the TEMPORARY password provided in the secure message
  3. Create a new password (minimum 14 characters)
  4. Important: Write down this password - it will be used for all SLAC-related systems

Step 3: Access Your Email

  1. Go to:
    https://o365.slac.stanford.edu/
  2. Log in with your username and the new password you just created

VPN Setup (Required for Internal Resources)

Prerequisites for VPN Access

Before setting up the VPN, ensure you have:

  • SLAC Windows account enabled for VPN access
    Request VPN access: https://www-internal.slac.stanford.edu/comp/windows/vpn/request/
  • Duo Two-Factor Authentication (2FA) enrolled
    Enrollment: https://www-internal.slac.stanford.edu/twostep/

Method 1: Automatic Installation (Recommended)

  1. Navigate to:
    https://vpn.slac.stanford.edu/
  2. Log in with your SLAC Windows credentials (do NOT prefix with "SLAC\")
  3. System will automatically detect your operating system and offer the appropriate client
  4. Click "Download" - the VPN client will install automatically
  5. Accept any security prompts during installation

Method 2: Manual Installation

If automatic installation fails or you need manual control:

Windows

  1. Download installer from https://vpn.slac.stanford.edu/
  2. Right-click installer â†’ "Run as Administrator" (critical step)
  3. Follow installation prompts
  4. Launch "Cisco AnyConnect Secure Mobility Client" from Start menu

Mac

  1. Download installer from https://vpn.slac.stanford.edu/
  2. Run installer (system may require security permissions)
  3. Allow system extensions in System Preferences > Security & Privacy
  4. Launch from Applications > Cisco folder

Connecting to VPN

  1. Launch Cisco AnyConnect/Secure Client
  2. Enter server: vpn.slac.stanford.edu
  3. Click Connect
  4. Enter credentials:
    • Username: Your SLAC Windows username
    • Password: Your SLAC Windows password
  5. Complete Duo 2FA when prompted
  6. Accept banner/terms if displayed
  7. Verify connection - client icon appears in system tray/menu bar

Troubleshooting VPN Issues

Connection Problems

  • Ensure using vpn.slac.stanford.edu (not other variations)
  • Disable conflicting VPN software

Authentication Issues

  • Verify Duo 2FA is properly enrolled
  • Ensure account has VPN access enabled
  • Check credentials work on other SLAC systems first

Training Portal (VPN Required)

Once connected to the SLAC VPN, you can view your assigned trainings:

URL Format:

https://www-internal.slac.stanford.edu/esh-db/training/slaconly/bin/show_training.asp?slac_key=YOUR_SLAC_ID

Example: If your SLAC ID is 123456, visit:

https://www-internal.slac.stanford.edu/esh-db/training/slaconly/bin/show_training.asp?slac_key=123456

This will display all trainings assigned to your account.

Troubleshooting

  • Ensure you're using the exact username and temporary password from the secure message
  • Password must be at least 14 characters
